1. Дан код:
var a = 1, b = 1, c, d;
  1. c = ++a; alert(c); // 2
  2. d = b++; alert(d); // 1
  3. c = (2+ ++a); alert(c); // 5
  4. d = (2+ b++); alert(d); // 4
  5. alert(a); // 3
  6. alert(b); // 3
Почему код даёт именно такие результаты?
  1. a было один, с префиксным инкрементов сначала увеличился на один и после использовалось значение
  2. b было один, его сначение сначала сохранилось в переменную d и после увеличилось на один постфиксным инкрементом
  3. в a было значение 2 после операции 1., значение a сначала увеличивается на 1, а потом используется в сложение с 2 из чего получаем 2 + 3 = 5
  4. в b было значение 2 после операции 2., значение b сначала используется в сложении с 2 из-за чего в d ложится значение 2 + 2 = 4, и после значение b увеличивается на 1 постфиксным инкрементом
  5. после операции 3. в a находится значение 3
  6. после операции 4. в b тоже находится 3
2. Дан код:
var a = 2;
var x = 1 + (a *= 2);

это интересно что не вылезет ошибка и значение a используется после умножения на 2 в операции сложения с 1 и в x запишется 5
